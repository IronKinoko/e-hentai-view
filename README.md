# e-hentai-view

A desktop App for ExHentai, build with Next.js and Electron

## How to use

```shell
# clone this repo
git clone https://github.com/IronKinoko/e-hentai-view.git

# yarn install
yarn

# run build
yarn run dist

cd dist

#open EhentaiView Setup.exe
```

## development

```shell
# The first time
yarn run build-electron

yarn start
```
